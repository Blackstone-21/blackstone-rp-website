# Blackstone RP Website v4.6 — Shop Management

- Removed **Features** from the top navigation and replaced it with **Shop**.
- Added a responsive public shop section to the homepage.
- Added a new **Shop** section to the admin panel.
- Added create, edit, publish, feature, sold-out and delete controls for shop items.
- Added `shop.manage` to role permissions. Founder and Administrator roles receive it automatically through a safe database migration.
- Added shop products to public API data, dashboard counts and full admin exports.
- Enforced HTTPS for external shop image and purchase links.
- Existing members, roles, applications, departments, announcements and settings are preserved.
