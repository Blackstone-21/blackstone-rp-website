# Blackstone RP Website v4.8 — Tebex Store

- Added a configurable official Tebex store link to the public Shop section.
- Added Tebex store URL and visibility controls to Admin → Settings.
- Added a direct link from the admin panel to the Tebex Creator Panel.
- Updated shop product fields to accept individual Tebex package links.
- Existing manual product cards remain available for featured packages and promotions.

## Setup
1. Create or open your Tebex store.
2. In the website Admin Panel, open Settings.
3. Paste the full HTTPS Tebex store URL and enable the Tebex store.
4. Add packages in Tebex, then optionally add featured cards in Admin → Shop using each package URL.
